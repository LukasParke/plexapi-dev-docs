# Introduction

Welcome to the PlexAPI.dev developer documentation. This site helps you integrate with the Plex Media Server API, understand its data model, and use community SDKs and tools.

## What is PlexAPI.dev?

PlexAPI.dev is a community initiative to document, simplify, and standardize building applications on top of Plex Media Server. Whether you are writing a custom dashboard, a metadata enrichment tool, or a media automation script, you will find practical guides and reference material here.

## What you will find here

- **Guides** — task-oriented walkthroughs for common integration scenarios.
- **Reference** — endpoint and schema documentation for the Plex API.
- **SDKs** — installation and usage docs for language-specific client libraries.

## Prerequisites

Most examples assume:

- A running Plex Media Server you can access.
- A Plex authentication token (see [Authentication](./authentication)).
- Basic familiarity with REST APIs and JSON.

# SDKs

PlexAPI.dev maintains and links to SDKs that wrap the Plex Media Server API in idiomatic code.

## Official and community SDKs

| Language | Package | Repository | Status |
|----------|---------|------------|--------|
| Python | `python-plexapi` | [pkkid/python-plexapi](https://github.com/pkkid/python-plexapi) | Community maintained |
| TypeScript | `@plexapi/sdk` | [plexapi-dev/docs](https://github.com/plexapi-dev/docs) | In development |

## Design goals

- Hide incidental HTTP complexity behind language-native APIs.
- Stay aligned with a shared, versioned API contract.
- Provide typed models where the language ecosystem supports it.

## Request a language

If you'd like to see an SDK in another language, open a discussion in the docs repository.
